## 프로젝트 개요

Gazebo 시뮬레이터에서 3D LiDAR 센서로 드론을 탐지하고, ROS2를 통해 Unity 3D로 실시간 전송하여 THAAD 스타일 발사대가 자동으로 추적·요격하는 방공 시뮬레이션 시스템입니다.

---

## 시스템 구성

```
[Gazebo Harmonic]                [ROS2 Jazzy / WSL2]              [Unity 3D / Windows]
  3D 월드 + 드론 3대   →   PointCloud2 처리 + 클러스터링   →   타겟 시각화 + 자동 요격
  CPU LiDAR 16ch×360°     /detected_targets (PoseArray)       THAAD 발사대 + 미사일
                           ros_tcp_endpoint (TCP:10000)        레이더 HUD + 3개 카메라
```

---

## 주요 기능

**LiDAR 처리 (Python / ROS2)**
- Gazebo 3D LiDAR → PointCloud2 수신
- 복셀 기반 배경 제거 (정적 지형 필터링)
- Greedy 클러스터링으로 드론 탐지
- `/detected_targets` PoseArray 토픽 퍼블리시 (10Hz)

**Unity 시각화 및 요격**
- ROS-TCP-Connector로 ROS2 토픽 실시간 수신
- 탐지된 드론을 3D 공간에 자동 생성·추적
- THAAD 발사대 Yaw/Pitch 자동 조준 (예측 조준 포함)
- 미사일 발사 → 호밍 유도 → 폭발 이펙트
- 레이더 HUD, 조준 카메라(AIM VIEW), 전체 조망 카메라(OVERVIEW)
- 요격 이벤트 `/target_destroyed` ROS2 퍼블리시

## 실행 방법

```bash
# 1. WSL — Gazebo 실행
cd ~/gazebo_ws && source /opt/ros/jazzy/setup.bash && source install/setup.bash
ros2 launch air_defense_sim sim.launch.py

# 2. WSL — ROS-TCP 엔드포인트 실행 (별도 터미널)
source /opt/ros/jazzy/setup.bash
ros2 run ros_tcp_endpoint default_server_endpoint --ros-args -p ROS_IP:=0.0.0.0

# 3. Unity — AirDefense > Build Scene → Play
#    Robotics > ROS Settings에서 WSL IP 입력
```
